Bonjour,

Vous souhaitez faire un essai du site du synthétiseur ? Ce sera avec plaisir !

Pour ce faire c'est très simple, il vous suffit de cliquer sur le bouton Démarrer l'essai.
Sinon vous pouvez cliquer sur le bouton Annuler pour revenir à la page d'accueil du site.

Si vous choisissez de faire une essai cela vous ouvrira une session invité.
En tant qu'invité vous pourrez faire les mêmes choses qu'un utilisateur normal mais avec les limitations suivantes :

- nombre maximal d'articles : 5 (une fois le quota atteint un message s'affichera, message qui disparaitra une fois un article supprimé) ;
- pas de posibilité de visualiser/éditer le profil utilisateur ;
- ... c'est tout ^^ !

A noter que j'héberge le site sur mon ordinateur personnel, qui n'est pas vraiment une "machine de guerre". C'est pourquoi je ne lancerai le serveur que pendant des horaires "normaux" (par exemple 9h - 18h).

Cela explique également le fait que la recherche d'articles par mots clés, qui fonctionne très bien sur le serveur de développement (si si je vous assure, j'ai testé !), n'est pour le moment pas mise en place sur le serveur de production (Elasticsearch étant très gourmand en ressources).

J'ai pour projet, dans un futur relativement proche, d'héberger ce site sur un Raspberry Pi doté de suffisament de mémoire pour permettre de lever cette limitation.

Merci pour votre compréhension et bonne utilisation :-)

L'équipe de développement, a.k.a. Julien
